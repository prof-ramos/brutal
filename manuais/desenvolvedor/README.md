2. Entenda a [Integração com o CMS](02-verificacao-cms.md)
3. Implemente [Deploy no Vercel](03-vercel-deployment.md)

- **Dúvidas sobre o CMS:** Consulte o [Manual da Admin Page](01-admin-page-manual.md)
